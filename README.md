# MotionForge-Colab

🚀 Free Google Colab Motion Transfer Toolkit

MotionForge-Colab is an open-source Google Colab project that allows users to generate AI videos by transferring motion from a reference video to a character image.

## Features

- Character Image → Animated Video
- Reference Motion Transfer
- Google Colab Support
- One Click Setup
- Free & Open Source
